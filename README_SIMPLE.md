# App Nutricional Simple 🍎

Una app súper simple para contar calorías, sin complicaciones.

## ✨ Características:

- ✅ Agregar alimentos con calorías
- ✅ Ver total de calorías del día
- ✅ Lista de todos los alimentos agregados
- ✅ Interfaz limpia y fácil de usar

## 🚀 Cómo usar:

1. Escribe el nombre del alimento
2. Escribe las calorías
3. Presiona "Agregar Alimento"
4. ¡Listo! Ve tu total actualizado

## 🛠️ Para compilar:

1. **Instala Android Studio** desde https://developer.android.com/studio
2. **Edita local.properties** con la ruta de tu SDK:
   ```
   sdk.dir=/ruta/a/tu/Android/Sdk
   ```
3. **Abre el proyecto** en Android Studio
4. **Ejecuta** la app

## 📱 Características técnicas:

- **Mínimo Android 7.0** (API 24)
- **Sin base de datos** - todo en memoria
- **Sin internet** - funciona offline
- **Código muy simple** - fácil de entender

## 🎯 Funcionalidad:

La app es perfecta para:
- Contar calorías básico
- Aprender desarrollo Android
- Tener algo que funcione rápido
